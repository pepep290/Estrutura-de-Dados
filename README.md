TP_1 - Projeto que funciona como um corretor de jogo da velha, recebendo a entrada dos resultados dos jogos e determinando se foi vitória de algum símbolo, empate ou de quem seria a próxima jogada, caso seja um jogo ainda imcompleto.
compilar e executar:
gcc *.c -o Tp1
./Tp1




TP_2 - Projeto que implementa diferentes tipos de pesquisa externas, Acesso Sequencial Indexado, Árvore Binária externa, Árvore B e Árvore B*, assim como também uma análise de desempenho, com comparações, leituras e escritas no disco
compilar e executar:
gcc *.c -o Tp2
./Tp2




TP_3 - Trabalho que implementa três tipos de ordenação externa, intercalação balanceada de 2f caminhos, intercalação balanceada de f+1 caminhos e quicksort externo, com uma análise de desempenho no final da execução
compilar e executar:
gcc *.c -o Tp3
./Tp3
